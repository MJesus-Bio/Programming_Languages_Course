# Programming Languages en Coursera
Este repositorio contiene mis resoluciones para los ejercicios del curso [Programming Languages](https://www.coursera.org/learn/programming-languages) dictado en la plataforma digital Coursera por la Universidad de Washington.
El curso, dividido en tres secciones, aporta una base sólida sobre aspectos fundamentales de la programación, con énfasis en programación funcional (aunque la tercera sección abarca Programación Orientada a Objetos). 

[Reseña del curso](https://vkontech.com/course-review-programming-languages-series-on-coursera/)
